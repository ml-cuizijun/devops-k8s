# Kubernetes 学习目录（高频实战版）

这套目录按「先能用 -> 能排障 -> 能交付」来设计，优先覆盖工作高频场景。

## 建议目录结构

```text
k8s核心清单/
├── README.md
├── 00-学习路线与方法.md
├── 01-核心概念与对象.md
├── 02-应用发布与流量入口.md
├── 03-存储与有状态服务.md
├── 04-安全与权限控制.md
├── 05-可观测性与故障排查.md
├── 06-调度弹性与高可用.md
├── 07-Helm_Kustomize与多环境.md
├── 08-CI_CD与GitOps.md
└── 09-实战打卡清单.md
```

## 阅读顺序

1. `00-学习路线与方法.md`
2. `01-核心概念与对象.md`
3. `02-应用发布与流量入口.md`
4. `03-存储与有状态服务.md`
5. `05-可观测性与故障排查.md`
6. `06-调度弹性与高可用.md`
7. `07-Helm_Kustomize与多环境.md`
8. `08-CI_CD与GitOps.md`
9. `09-实战打卡清单.md`

## 使用建议

- 每个章节都做一次最小可运行实验。
- 每周至少做一次「故障复盘」。
- 把自己项目的 YAML 贴进对应章节做对照学习。
